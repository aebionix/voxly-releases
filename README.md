# Wisper App - Releases

Offizielle Releases der Wisper App - Lokale Spracherkennung mit Whisper.

## Download

**Aktuelle Version:** Siehe [update-info.json](update-info.json)

**Download:** [Releases-Ordner](releases/)

## Installation

1. ZIP herunterladen aus `releases/`
2. An gewünschten Ort entpacken (z.B. `%LOCALAPPDATA%\WisperApp`)
3. `WisperApp.exe` starten

### Voraussetzungen

- Windows 10/11
- .NET 10 Runtime
- FFmpeg (im PATH)
- whisper.cpp Binary + Model

Die App prüft automatisch auf Updates via `update-info.json` in diesem Repository.
